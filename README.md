# AlienousX – AI Assistant for TRCAC

An AI-powered assistant designed to help students of **Thakur Ramnarayan College of Arts and Commerce (TRCAC)** quickly access information about the college such as admissions, courses, campus life, and student services.

This project is created **purely for educational purposes** to demonstrate how AI agents can improve the student information experience.

---

# Overview

Students often need information about:

- Admissions process
- Courses and programs
- Campus facilities
- College events
- Student services

Usually, students must **scroll through multiple pages of the college website** to find answers.

**AlienousX solves this problem.**

Instead of searching manually, students can simply **ask the AI assistant**, which is trained on a **custom dataset related to TRCAC**, and receive instant answers.

The assistant provides a **faster and more interactive way** for students to get the information they need.

---

# Purpose

The goal of this project is to demonstrate:

- How AI agents can assist students in educational institutions
- How natural language interaction improves information access
- How AI can reduce the need for manual browsing through websites

This system is designed **only for educational and demonstration purposes**.

---

# Problem Solved

Traditional college websites require students to:

- Navigate through multiple menus
- Search different pages
- Spend time finding simple information

**AlienousX improves this by allowing students to:**

- Ask questions directly
- Get instant answers
- Avoid scrolling through the entire website

### Example questions students can ask:

- What courses are available at TRCAC?
- How do I apply for admission?
- What facilities are available on campus?
- What events are happening in the college?

---

# How This Agent Helps TRCAC Students

AlienousX helps students by:

- Providing **instant answers about college information**
- Acting as a **virtual campus guide**
- Helping new students understand **admission procedures**
- Explaining **courses and academic programs**
- Assisting with **campus life and student activities**

Instead of browsing many pages, students can **directly interact with the AI assistant**.

---

# Tech Stack

The project is built using simple and modern web technologies:

- HTML5
- CSS3
- JavaScript
- Chatbase AI integration
- Custom AI dataset for TRCAC information
- GithHub (Deployment)

The AI chat interface is embedded using **Chatbase for real-time interaction**.
